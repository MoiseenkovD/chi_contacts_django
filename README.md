## Clone project
```shell
$ git clone https://github.com/MoiseenkovD/chi_contacts_django.git
```

## Install
```shell
$ cd chi_contacts_django
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install django
$ pip install djangorestframework
$ python3 manage.py migrate
```

## Run 
```shell
$ python3 manage.py runserver
```
